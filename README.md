# takken-gakushu-site

宅建士試験の過去問演習・復習・弱点対策・講座比較サイトです。

## 概要

Next.js App Router と TypeScript で構築する、宅建士専用の学習サイトです。
宅建過去問JSONを読み込み、年度別演習・全年度ランダム演習・間違えた問題の復習を行えるようにしています。

## 作成済みページ

- `/` : トップページ
- `/past` : 宅建過去問一覧ページ
- `/practice` : 宅建過去問演習ページ
- `/review` : 復習ページ
- `/traps` : ひっかけ対策ページの準備ページ
- `/weakness` : 弱点対策ページの準備ページ
- `/courses` : 宅建講座比較ページの準備ページ
- `/privacy` : プライバシーポリシー
- `/terms` : 利用規約
- `/contact` : お問い合わせ案内ページ

## 開発コマンド

```bash
npm install
npm run dev
npm run build
```
